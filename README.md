# Issue Companion — RAG-помощник по заявкам и инструкциям

## Проблема
Сотрудники технической поддержки тратят часы на поиск информации в старых заявках и инструкциях

## Решение
RAG-система, которая находит релевантные документы и решенные заявки, и генерирует ответ вместе со списком источников

## Архитектура
[Простая схема: Документы → Chunking → Embeddings → Qdrant → Retrieval → LLM → Ответ]

## Быстрый старт
```bash
docker-compose up
```
